## Commandes

* `mkdocs new [dir-name]` - Créer un nouveau projet.
* `mkdocs serve` - Démarrer un serveur avec rafraîchissement automatique.
* `mkdocs build` - Générer le site de documentation.
* `mkdocs -h` - Afficher l'aide et s'arrêter.
